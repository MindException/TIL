# Django


```bash
django-admin startproject start_django [경로명]
```
* cmd 창에 위와 같이 입력하여 장고 프로젝트를 생성한다.

## Django 폴더 구조

```css
start_django/
--->start_django/
    --->__init__.py
    --->settings.py
    --->urls.py
    --->wsgi.py
--->manage.py
```

1. __start_django__ : Project 파일이 포함된 디렉토리로 모든 파일을 담는 용도 외에는 사용되지 않는다.

2. __start_django__ : Django 프로젝트 수행을 위한 패키지가 저장된 프로젝트이다.
